# News Project

## How to initialize

- Open the main folder
- Open terminal
- generate .env
- configure your .env as .env.example
- run npm/yarn to install the modules
- run npm/yarn run migration:run
- run npm/yarn run dev

## Routes

- User

  - post
  - get
  - get/:id
  - delete/:id
  - patch/:id

- Category

  - post
  - get
  - delete/:id
  - patch/:id

- News
  - post
  - get
  - get/:id
  - delete/:id
  - patch/:id
